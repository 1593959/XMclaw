## 自动进化流程

### 每小时 HEARTBEAT 执行

```
1️⃣ 观察阶段 (Observer)
   ├─ 分析对话历史（从 CoPaw 获取）
   ├─ 检测用户行为模式
   ├─ 识别反复出现的问题类型
   └─ 发现新的任务类型

2️⃣ 学习阶段 (Memory)
   ├─ 短时：保存本次观察
   ├─ 中时：与历史对比，发现趋势
   └─ 长时：更新知识库

3️⃣ 进化阶段 (Evolution)
   ├─ 检查是否满足 Gene 生成条件
   ├─ 自动创建新 Gene
   ├─ 更新能力树
   └─ ADL 安全检查

4️⃣ 验证阶段 (Validation)
   ├─ 小范围测试新能力
   ├─ 监控 V-Score 变化
   └─ 自动回滚（如有问题）
```

## Gene 自动生成条件

```javascript
{
  // 模式重复条件
  pattern_repeat: 3,        // 同一模式出现 3 次
  
  // 任务复杂度条件
  task_complexity: 'medium', // 复杂度中等
  
  // 频率条件
  frequency_threshold: 0.7,  // 70% 对话涉及
  
  // 新颖性条件
  novelty_score: 0.6,        // 与现有 Gene 重叠度 < 60%
}
```

```json
{

    "enabled": true,
    "observation_interval_min": 30,
    "evolution_trigger_threshold": 3,
    "auto_gene_creation": true,
    "auto_skill_creation": true,
    "safety": {
      "require_validation": true,
      "rollback_on_regression": true,
      "max_genes_per_day": 5
    }
  },
  "pattern_detection": {
    "min_occurrences": 3,
    "similarity_threshold": 0.7,
    "forbidden_patterns": ["password", "secret", "key"]
  }
```

## CLI 命令

```bash
# 启动自动进化系统
node xm-auto-evo/index.js start

# 单次观察分析
node xm-auto-evo/index.js observe

# 生成进化建议
node xm-auto-evo/index.js suggest

# 查看三层记忆状态
node xm-auto-evo/index.js memory status

# 手动触发进化
node xm-auto-evo/index.js evolve

# 查看系统状态
node xm-auto-evo/index.js status
```
